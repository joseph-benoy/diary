# Personal Diary

Day-to-day personal diary built using Bootstrap , React Js , express Js and MongoDB

### Usage
To run the both the server and client
```
npm start
```

To run server for dev
```
npm run start-dev
```

To run client ui only
```
cd client
npm start
```
